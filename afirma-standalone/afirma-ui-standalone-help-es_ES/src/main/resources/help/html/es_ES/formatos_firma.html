Formatos de firma electrónica

Los formatos de firma son especificaciones, comúnmente aprobadas como estándares reconocidos, que definen qué información debe o puede contener una firma electrónica y el cómo se estructura esa información.

Existen numerosos formatos de firma (CAdES, XAdES, PAdES, ODF,.) y, de forma genérica, se podrían diferenciar entre formatos de firma binaria y firma XML, según organicen la información en una estructura ASN.1 o en una estructura XML, respectivamente.
